%d: Utilizado para imprimir enteros con signo (int).
%u: Utilizado para imprimir enteros sin signo (unsigned int).
%f: Utilizado para imprimir números de punto flotante (float).
%lf: Utilizado para imprimir números de punto flotante de doble precisión (double).
%c: Utilizado para imprimir caracteres (char).
%s: Utilizado para imprimir cadenas de caracteres (strings).
%x o %X: Utilizado para imprimir enteros en formato hexadecimal.
%o: Utilizado para imprimir enteros en formato octal.
%p: Utilizado para imprimir punteros.
%e o %E: Utilizado para imprimir números de punto flotante en notación científica.
%g o %G: Utilizado para imprimir números de punto flotante en notación decimal o científica, dependiendo de la precisión.
